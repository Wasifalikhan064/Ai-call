## Usage

### To Install and Test

```bash
git clone git@github.com:Wasifalikhan064/Ai-call.git
cd Hood
```

```bash
# Install server dependencies
cd backend
npm install

# Install client dependencies
cd frontend
npm install

```

```bash
# Start the server
cd backend
npm start

# Start the client
cd frontend
npm run dev

```
